## PDF/Textbook viewers
For PDF/Textbook viewing and note taking you can try:
- *Samsung Notes* uploading each textbook there were you can highlight with the pen and tablet... may have to sign into your own account... but we can sort that out if you choose this option :)
- [Xodo](https://apps.microsoft.com/detail/9WZDNCRDJXP4?hl=en-us&gl=ZA&ocid=pdpshare) can easily add a folder that contains all your textbooks/pdfs. Can do all sorts of editing and highlights. I like this. *(Free, no sign in)*
- [Drawboard PDF](https://us-prod-prod.msix.dbpdf.drawboard.com/downloads/Db.App.appinstaller?utmParams=msclkid:none,gclid:none,utm_source:none,utm_medium:none,utm_campaign:none,utm_content:none,utm_term:none) free but paid for best features. Very clean modern interface, can load documents from cloud storage. (Sign up required, if you dont mind paying then use this)
- OR go back to the stone age and use Edge or Firefox as a PDF viewer... 

## Extras (look at after next week **ONLY**)
### Vault Backups:
- I used to pay for obsidian sync which is $5 per month, it is basically a cloud backup of your vault that easily syncs, even across devices. 
- Alternatively you can upload your vault to GitHub (publicly) and even use the git plugin for document history, nice but probably not needed. Vault snapshots are set to recover notes every 5mins with a history of the last 7 days.
- The Remote Save plugin can be set up with your Dropbox and save your vault there. I have not used or tested this as I dont have Dropbox... this is free but may break IDK 'I dont know'.

### Charts and Graphics
- download or use online https://app.diagrams.net/ (has a janky obsidian plugin as well) 
- Obsidian *canvas* `Ctrl + P ... canvas`
- *Kanban* plugin `Ctrl + P ... kanban` or create in the `Boards etc` folder
- *Excalidraw* plugin in obsidian or https://excalidraw.com/ to use online... is a bit janky but I like that I can `[[like to note]]` in my drawings.
- *Charts* plugin in obsidian, if you want to make pretty bar or line graphs

###  PDF tools continued
==Note: You can use obsidian for PDFs but I dont like it, thus recommend you to use a separate app for it :) ==

Other PDF tools you can look into *when bored*:
- *Zotero*: best for saving webpages, journal articles, and PDFs. (Focus on referencing source) Although its more 'academic' in nature, it may be a handy tool.
- *Calibre*: awesome for viewing collection like books on a shelf. I like their tags feature. You can also convert from epub to PDF with ease.  

Free PDF editor tool: https://www.pdfgear.com/
